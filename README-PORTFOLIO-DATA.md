# Portfolio Data JSON

This repository contains all portfolio data in JSON format for easy consumption and integration.

## File: `portfolio-data.json`

This file contains structured data extracted from the portfolio website including:

### Data Structure

```json
{
  "profile": {
    // Personal information, bio, and images
  },
  "education": [
    // Educational background with degrees, institutions, and GPAs
  ],
  "interests": [
    // Areas of professional interest
  ],
  "experiences": [
    // Work experience and roles
  ],
  "skills": [
    // Technical skills organized by category with technologies
  ],
  "projects": [
    // Portfolio projects with descriptions, links, and tech stacks
  ],
  "achievements": {
    // HackerRank ratings, certifications, and competitive programming accomplishments
  },
  "contact": {
    // Contact information and form endpoint
  },
  "socialLinks": [
    // Social media and professional platform links
  ],
  "metadata": {
    // Copyright and last update information
  }
}
```

### Contents Summary

- **Profile**: Name, title, bio, and profile images
- **Education**: 3 degrees (BSc, HSC, SSC)
- **Skills**: 6 skill categories with 19 total technologies
- **Projects**: 6 complete projects with live demos and source code
- **Achievements**: HackerRank 5-star ratings, 7 certifications, competitive programming highlights
- **Social Links**: 6 platforms (GitHub, LinkedIn, Facebook, Instagram, Codeforces, LeetCode)
- **Contact**: Email and contact form information

### Usage

You can use this JSON file to:
- Build alternative frontend interfaces
- Create mobile applications
- Generate résumés and CVs
- Integrate with portfolio management systems
- Feed data to APIs or databases
- Create data visualizations

### Validation

The JSON file has been validated and conforms to proper JSON standards.
