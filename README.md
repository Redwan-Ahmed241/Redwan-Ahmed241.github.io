## Run Locally

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

Optional (recommended):

1. Use VS Code Live Server extension.
2. Run with "Open with Live Server" for instant refresh while editing.

## Project Structure

- `index.html` - Main page layout and content
- `styles.css` - Site styling and responsive behavior
- `script.js` - Navigation behavior and form interactions
- `profile.jpg` / `about-pic.jpg` - Portfolio images
- `cv.tex` - CV source file
